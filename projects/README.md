# Projects

Use `projects/_template.html` as the starter for new project pages.

Suggested workflow:

1. Copy `projects/_template.html`.
2. Rename it, for example `projects/my-project.html`.
3. Edit the title, sections, GitHub link, PDF path, and download links.
4. Add or update the matching project entry in `data/site-data.json`.

Project cards support:

- `href`: internal project page path
- `links`: optional external links such as GitHub, paper, demo, or docs
